# prettier-configure

Setting up for [Prettier](https://github.com/prettier/prettier) and [Husky](https://github.com/typicode/husky) automatically.

## Usage

```sh
npx prettier-configure
```

## `.prettierrc.yaml`

| rule            | value  |
| --------------- | ------ |
| `trailingComma` | `all`  |
| `tabWidth`      | `4`    |
| `singleQuote`   | `true` |

## `.prettierignore`

```
/node_modules
package.json
```
